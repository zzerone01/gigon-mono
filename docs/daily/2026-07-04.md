### PostHog 퍼널 계측 (apps/app + API)

랜딩(gigon.io)에만 있던 PostHog를 서비스에 확장. **핵심 퍼널은 서버사이드 캡처** —
모든 상태 전이가 app.gigon.io/api 로만 지나가므로(웹·모바일 공통) 라우트 핸들러에서
`defer(() => track(...))` 한 줄로 클라이언트 종류와 무관하게 전체 퍼널이 잡힌다.

- **서버**: `lib/server/analytics.ts` — posthog-node `captureImmediate`(서버리스에서
  배치 유실 없음). 키 없거나 vitest면 no-op. distinct_id = Supabase user id
- **클라**: `app/providers.tsx` — posthog-js 페이지뷰/오토캡처. 로그인하면 같은
  user id로 identify → 서버 이벤트와 한 사람으로 합쳐짐. `/ingest` 리버스 프록시
  (next.config.js rewrite). ⚠️ proxy.ts 매처에서 `ingest` 제외 필수 — 안 하면
  로그아웃 상태(로그인 페이지) 이벤트가 /login 307로 유실됨
- **모바일**: SDK 안 붙임(의도) — 퍼널 이벤트는 API에서 잡히므로 화면 뷰만 없음.
  필요해지면 posthog-react-native 추가

이벤트 스키마 (PRD §8 KPI: 공고→지원→매칭→도착→PIN완료):

| 이벤트 | 발화 지점 | distinct_id |
| --- | --- | --- |
| `gig_viewed` | 워커 상세 시트 열림 (클라이언트) | 워커 |
| `gig_posted` · `gig_cancelled` | POST /api/gigs · [id]/cancel | 사장님 |
| `gig_applied` | /api/gigs/[id]/apply | 워커 |
| `match_created` | /api/applications/[id]/select | **양쪽** (prop `role`: worker/business) |
| `worker_arrived` | /api/matches/[id]/arrive | 워커 |

## 사용자 액션 필요 (스토어 제출 전)

1. **Google Play Console 개발자 계정** ($25 일회성, play.google.com/console) —
   신원 확인이 며칠 걸릴 수 있어 **가장 먼저 시작할 것**. 계정 주체(개인/조직) 결정
2. **Firebase 프로젝트** (FCM — Android 푸시 수신에 필수):
   console.firebase.google.com → 프로젝트 생성 → Android 앱 추가(패키지 `com.gigon.app`)
   → `google-services.json` 다운로드 → `apps/mobile/`에 저장 (gitignore 됨)
   → 프로젝트 설정 > 서비스 계정 > 새 비공개 키 생성(JSON)
   → `cd apps/mobile && npx eas-cli credentials` → Android → Push Notifications →
   FCM V1 서비스 계정 키 업로드
3. **Android용 Google Maps 키**: 웹 키는 referer 제한이라 네이티브에서 안 됨.
   Google Cloud Console에서 새 키 생성 → "Maps SDK for Android" 활성화 →
   Android 앱 제한(패키지 `com.gigon.app` + EAS 키스토어 SHA-1) →
   `npx eas-cli env:create --name GOOGLE_MAPS_ANDROID_API_KEY --value <키>` (또는 eas.json env)
4. 2·3 완료 후: `cd apps/mobile && npx eas-cli build -p android --profile production`
   → `eas submit -p android` (Play Console 연결)
5. Play Console 스토어 등록정보: 스크린샷, 개인정보처리방침 URL(https://app.gigon.io/privacy),
   데이터 보안 양식
