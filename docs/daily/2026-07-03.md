### 5. 트러블슈팅 기록 (재발 방지)

| 문제 | 해결 |
| --- | --- |
| PostgREST 404/permission denied | 테이블에 `authenticated` GRANT 필요 (마이그레이션 `…010000`) |
| `gen_salt` not found | pgcrypto는 `extensions` 스키마 → PIN 함수 `search_path = public, extensions` |
| 로컬 phone OTP `phone_provider_disabled` | 더미 Twilio provider 활성화 필요 (test_otp 번호는 실발송 안 함) |
| Colima에서 vector 컨테이너 실패 | `[analytics] enabled = false` |
| Next 16 middleware deprecated | `proxy.ts` + `export function proxy` |
| 127.0.0.1 dev HMR 차단 | `allowedDevOrigins: ["127.0.0.1"]` |
| pnpm+Metro 모듈 해석 실패 | `disableHierarchicalLookup` 제거, `@expo/metro-runtime` 직접 의존성 |
| `next build`를 dev 서버 실행 중 돌리면 `.next` 충돌 | 동시에 돌리지 말 것 |

Twilio PH SMS가 $0.17/건으로 너무 비싸서 **필리핀 로컬 Semaphore**(₱0.56/건, OTP 라우트
₱1.12)로 결정. Supabase **Send SMS Hook**으로 연동 완료:

- `supabase/functions/send-sms` 엣지 함수 (훅 서명 검증 → Semaphore `/api/v4/otp`)
- 비밀값은 `supabase/.env.local`(gitignored) + `supabase secrets set` — **레포에 키 없음**
- config.toml `[auth.hook.send_sms]` + `[functions.send-sms] verify_jwt=false`,
  클라우드 반영 완료(config push). 서명/무서명 프로브로 체인 검증됨
- 함수 배포는 `--use-api` 필요 (Colima에서 로컬 Docker 번들 실패)
- ⏳ 남은 건 Semaphore 계정 쪽: 프로필 승인(Pending) → **Sender Name 신청** → 크레딧 충전.
  전부 되면 실번호 로그인 즉시 활성화 (테스트 번호는 훅을 안 타므로 계속 동작)
